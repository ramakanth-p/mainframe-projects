\# Employee Account Batch Reporting System



\## Project Overview



The Employee Account Batch Reporting System is a batch application developed in an IBM z/OS mainframe environment using COBOL and DB2.



The system demonstrates a typical enterprise batch processing workflow where employee account data is extracted from a DB2 table, written to a sequential file, and then processed to generate a formatted report.



The application is implemented using two COBOL programs executed through JCL jobs.

\## Program Description



\### EABR01 вЂ“ DB2 Data Extraction Program



Responsibilities:



\- Reads employee account data from DB2 table \*\*Z88436.Z88436T\*\*

\- Filters records where \*\*BALANCE > 0\*\*

\- Writes extracted records to sequential dataset \*\*EABRF01\*\*

\- Handles SQLCODE and file status error checks

\- Maintains processed record count



---



\### EABR02 вЂ“ Report Generation Program



Responsibilities:



\- Reads sequential dataset \*\*EABRF01\*\*

\- Formats numeric values for display

\- Generates report headers and column headings

\- Maintains pagination for multi-page reports

\- Calculates total records and total balance

\- Produces formatted report \*\*EABRF02\*\*
